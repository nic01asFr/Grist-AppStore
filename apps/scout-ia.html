Installation en 3 étapes
1

Télécharger

Appuyez sur le bouton ci-dessus. Le fichier .apk se télécharge sur votre téléphone.

2

Autoriser

Dans Paramètres Android → Sécurité → activer Sources inconnues pour votre navigateur.

3

Installer

Ouvrez le fichier et appuyez sur Installer. Lancez SCOUT IA depuis votre écran d'accueil.

Fonctionnalités
🎙

Enregistrement vocal

Commentaire audio en un tap, waveform animée, chronomètre.

📝

Transcription live

STT par chunks de 10s via SSPCloud (gemma4-31b) ou Albert. Texte progressif.

✦

Extraction LLM

Type, localisation, urgence, actions, matériaux — remplis par streaming.

📍

GPS automatique

Coordonnées capturées au démarrage. Pré-remplissage du champ localisation.

🔄

Sync Grist

Sauvegarde directe dans votre document Grist via API REST sécurisée.

🗂

Init automatique

Crée la table Visites_terrain avec colonnes typées et choix colorés si elle n'existe pas.
