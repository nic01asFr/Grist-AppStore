Android 7.0+ requis · Activer "Sources inconnues" pour installer
Installation en 3 étapes
1

Télécharger

Appuyez sur le bouton ci-dessus. Le fichier .apk se télécharge sur votre téléphone.

2

Autoriser

Dans Paramètres Android → Sécurité → activer Sources inconnues pour votre navigateur.

3

Installer

Ouvrez le fichier téléchargé et appuyez sur Installer. Lancez SCOUT IA depuis votre écran d'accueil.

Fonctionnalités
🎙

Enregistrement vocal

Commentaire audio en un tap, waveform animée, chronomètre.

📝

Transcription automatique

STT via SSPCloud (gemma4-31b) ou Albert API.

✦

Extraction par IA

Type, localisation, urgence, actions — remplis automatiquement.

🔄

Sync Grist

Sauvegarde directe dans votre document Grist via API sécurisée.

🗂

Init automatique

Crée la table Visites_terrain si elle n'existe pas. Mapping intelligent.

Données locales
